Dunge – Support

A living narrative — help, troubleshooting, and subscription support

Welcome to Dunge support. Below you'll find help for common tasks, guidance on LLM-privacy behavior (important), subscription details, and how to contact us for anything else.

Core Features

  • LLM-driven narrative: Full scenes, choices, and events are generated by a cloud LLM to keep the world consistent and reactive.
  • Event-based state: The game applies deterministic events locally (health changes, inventory) so state is reliable even though narration is generated.
  • Local-first saves: Character saves, inventory, and journals are kept on-device by default.
  • Subscriptions: Premium features and HD TTS are managed via RevenueCat (no raw card data stored by us).
  • No ads: Dunge does not show advertising.

Privacy & terms: Privacy Policy

Subscriptions (RevenueCat)

Dunge offers an optional subscription for premium features and HD TTS. Purchases are handled via RevenueCat and the App Store. Current tiers:

  • Premium — unlimited journeys and HD TTS.
  • Annual — yearly billing at a discounted rate.
  • Purchase: Tap Upgrade inside the app and complete checkout using Apple's payment sheet.
  • Restore purchases: Use "Restore Purchases" in Profile → Subscriptions; contact support if restore fails.
  • Manage / Cancel: Manage subscriptions via your Apple ID account settings.

LLM & Privacy — Important

Every in-game action — including preset choices — is sent to the LLM provider. This enables coherent, reactive storytelling but means short text summaries of state and recent context are transmitted to the model. See the Privacy Policy for full details.

  • What we send: compact state summary, the selected or typed action, recent narrative context, and system instructions.
  • What we don't send: payment card data, Apple/Google account credentials, or personal identity fields not required for gameplay.
  • Offline behavior: In limited offline scenarios the app may show cached responses but full interactive play requires network access.

Troubleshooting

  • App crashes / freezes: Force-quit and reopen. If persistent, reinstall the app. Attach crash logs when contacting support.
  • Network errors / LLM timeouts: Check connectivity. If the LLM provider is unreachable the app may retry or show a short error message — network is required for full play.
  • Audio / TTS issues: Check device volume and permissions. Use Normal TTS if HD TTS fails (Normal uses local/native TTS where available).
  • Missing premium features after purchase: Tap Restore Purchases. If that doesn't work, email support with the App Store receipt.
  • Lost local saves: If you see missing saves, check device backups (iCloud) or contact support promptly — avoid creating a new account which may overwrite local data.

FAQ

Is Dunge suitable for children?
Dunge includes fantasy themes and some narrative content that may not be suitable for very young children. We do not knowingly collect personal information from children under 13.
Are my conversations stored on a server?
Game state and saves are stored locally. However, the short context we send to the LLM provider for each turn is processed by that provider to generate narrative responses. We do not maintain a first-party server that stores full chat logs by default.
How can I opt out of cloud LLM calls?
Currently there is no local-only mode; disabling cloud LLM calls disables core game functionality. We are evaluating a future "local-only" mode and will document it here if implemented.
How do I transfer data to a new device?
Use device-level backups (iCloud) to migrate local app data. Contact support if you need assistance with migration options.
